# 📦 API Inventario

API REST para la gestión de inventario con autenticación JWT, desarrollada con FastAPI y PostgreSQL.

## 🚀 Descripción

Sistema backend que permite gestionar productos de un inventario mediante operaciones CRUD (crear, leer, actualizar y eliminar), con autenticación segura mediante JSON Web Tokens.

## 🛠️ Tecnologías

- **Python 3.12+**
- **FastAPI** - Framework web moderno y rápido
- **SQLAlchemy** - ORM para base de datos
- **PostgreSQL** - Base de datos relacional
- **JWT** - Autenticación con tokens
- **Pydantic** - Validación de datos
- **Uvicorn** - Servidor ASGI

## 📋 Funcionalidades

- Registro y autenticación de usuarios
- CRUD completo de productos
- Autenticación con JWT
- Documentación automática con Swagger
- Validación de datos
